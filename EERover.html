EERover

First year EEE group project

Imperial College London

The aim of this project is to design a remotely-controlled rover that can explore a distant planet and identify unusual minerals. Using the properties of the minerals and the rover is able to detect these properties and classify the deposits that it finds. The mineral types and their properties include radio, IR and acoustic signal emission.

The rover is controlled wirelessly and the signals are detected using infrared and radio sensors. Using the different modulating frequencies the web app that has been designed can differentiate between the different signals and determine the material type.
